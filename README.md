# Unick System

## Backend (Laravel)

Requirements: PHP 8.2+, Composer, Node 20+

Setup:

- cd `unick-backend`
- composer install
- cp .env.example .env
- php artisan key:generate
- php artisan migrate
- php artisan serve (serves on http://127.0.0.1:8000)

## Frontend (React)

Requirements: Node 20+

Setup:

- cd `unick-frontend`
- npm install
- npm start (serves on http://localhost:3000)
